# agent.kit

Portable prompts and skills for agentic coding. Works with any harness that
supports the [Agent Skills standard](https://agentskills.io) — pi, Claude Code, etc.

## Structure

```
prompts/     Reusable prompt templates
skills/      On-demand capability packages (future)
```

## Usage

Browse and install via [agent.library](https://github.com/johanfriis/agent.library):

```
/skill:library use commit
/skill:library use commit globally
```

Or add directly to your harness config.
